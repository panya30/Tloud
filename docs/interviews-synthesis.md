## Surprising Insights by Persona

### Nong Ploy (Online Seller)
- AI perceived as **MORE reliable** than human admins for consistency
- Supplement sellers see AI as **SAFER** for compliance (follows templates exactly)
- "Shopee response-time score" = hidden growth lever (algorithmic urgency)
- Previous chatbot trauma (ZWIZ, LINE bots) creates strong initial skepticism

### P' Ton (Restaurant Owner)
- **Ordering is the real business** — GrabFood commission arbitrage saves 200K+/mo
- Facebook Messenger is **as important as LINE** (41% flagged LINE-only as limitation)
- "Clone my personality" is an emotional trigger for solo operators
- Restaurant consultants are an untapped B2B2C distribution channel

### Pim (Salon/Spa Owner)
- **AI reduces embarrassment** for sensitive services (waxing, acne, weight loss)
- Mobile service providers (home visits) have **2x no-show pain** (lost slot + travel)
- Commission savings from Booking.com/Agoda = distinct value proposition
- Walk-in businesses need **queue estimation**, not appointment booking

### Kru Bank (Tutor)
- **"Teacher's Assistant" framing** is critical — AI as "ผู้ช่วยครู[ชื่อ]"
- AI must **NEVER** discuss child progress, wellbeing, or safety — hard boundary
- ~30% of small tutors **don't have LINE OA** — setup help = instant value
- Adjacent markets are huge: fitness coaches, study facilitators, speech therapists

## MVP Specification (Informed by 400 Interviews)

### Build First (Week 1-4)
1. **3-click LINE OA connection** — the hero experience
2. **Knowledge base upload** (photo + text, mobile-first)
3. **Thai AI FAQ auto-reply** — catalog-grounded, no hallucination
4. **Human handoff** with instant LINE notification (<5 sec)
5. **100% Thai UI** — zero English
6. **Real-time inventory/availability toggle**
7. **PromptPay monthly billing** — no credit card, no contract

### Build Next (Month 2-3)
8. Facebook Messenger support (unlocks restaurant segment)
9. Booking/appointment system (unlocks salon + tutor segments)
10. Payment reminders (unlocks tutor segment)
11. Brand voice customization (3 presets: formal, friendly, cute)
12. Basic analytics dashboard

### Build Later (Month 4-6)
13. Shopee/Lazada chat integration
14. Ordering capability (restaurant commission arbitrage)
15. Multi-location management
16. Workflow automation
17. API access for power users
18. Compliance/forbidden-phrase mode

## Go/No-Go Signals

### GO Signals (Strong)
- 71% average positive sentiment across all personas
- 69% would pay at least 499 THB/mo
- Universal pain point (repetitive FAQ) confirmed at 85%+ frequency
- 499 THB/mo pricing validated — perceived as "very cheap" vs alternatives
- No Thai-first competitor exists in this space
- Government tailwind (THB 25B AI investment, 200% tax deduction for SME digital)

### CAUTION Signals
- LINE-only may limit restaurant segment (41% want Facebook too)
- Accuracy is existential — one wrong answer = lost trust permanently
- 25% free-only users = need strong free→paid conversion funnel
- Previous chatbot failures create skepticism barrier in 10-15% of market
- Regulatory sensitivity for health/beauty/supplement verticals

### NO-GO Would Be
- If Typhoon LLM Thai accuracy is below 90% (test in technical spike)
- If LINE OA API costs make unit economics negative at 499 THB/mo
- If 3-click setup is actually impossible (LINE OA has onboarding friction)
